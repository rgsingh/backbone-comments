Backbone-Comments

Hello! This is a demo of a small Backbone app that manages a simple commenting system, like one you would find on a blog. Comments can be written in Markdown and emails are verified with Mozilla Persona. The only special dependencies are Moment (for formatting timestamps) and Markdown.js (for rendering comment markdown.)

There's no backend server to store the comments (for now), but it could easily be adapted to communicate with a server or to localStorage in your browser. In future releases, users will be able to edit and delete their comments and use a GitHub-style "zen" comment editor. Basically it's more exciting than a TODO list ;)

The app demonstrates some of the concepts and best-practices discussed in the upcoming book Backbone In Action from Manning Publications. Take a look at the source on GitHub; the master branch has a the most basic, non-authenticated version of the app, and the persona and gh-pages branches contain the code running on wookiehangover.github.com/backbone-comments.
